# Specification

## Summary
**Goal:** Deploy the Sagrai Village Digital Platform (frontend and backend canisters) permanently to the Internet Computer mainnet so it is publicly accessible at a stable ICP URL.

**Planned changes:**
- Deploy the backend Motoko canister to the Internet Computer mainnet with a stable canister ID.
- Deploy the frontend assets canister to the Internet Computer mainnet, served at a permanent ic0.app URL.
- Ensure Internet Identity authentication works on the mainnet deployment.
- Verify all existing features (crop marketplace, notices, gallery, orders, contact, admin panel) remain fully functional after deployment.

**User-visible outcome:** The Sagrai Village platform is permanently live on the Internet Computer mainnet and accessible to anyone via a stable public URL without requiring a local replica.
